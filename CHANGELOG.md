## [0.1.0] - 2026-02-24

### Features

- Multi-agent orchestration plugin for Claude Code with persona-based workflows
- Agent roster: `sisyphus` (opus, orchestrator), `hephaestus` (opus, autonomous executor), `atlas` (sonnet, coordinator), `prometheus` (opus, planner), `explore` (haiku, code search), `librarian` (sonnet, research), `oracle` (opus, architecture), `metis` (opus, pre-planning), `momus` (opus, plan review)
- Skill library: persona switches, planning (`plan`, `start-work`), continuation control (`ulw`, `ralph-loop`, `stop-continuation`, `cancel-ralph`), utilities (`handoff`, `selftest`)
- Hook pipeline: SessionStart persona injection, UserPromptSubmit ultrawork detection, PreToolUse destructive-command guard, Stop continuation blocker
- Two-layer prompt composition with static agent `.md` files and dynamic sections generated by `scripts/build_sections.py`
- Boulder state management for plan tracking across sessions (`.agent-kit/boulder.json`)
- Python runtime scripts (stdlib-only): `hook_router.py`, `build_sections.py`, `detect.py`, `sanitize.py`, `state.py`, `telemetry.py`, `prompt_version.py`
- Comprehensive test suite with frontmatter validation, hooks schema checks, cross-reference integrity, and persona detection tests
- CI validation workflow (`validate.yml`) running on every push and pull request
